Gigil's Russian Roulette

Een gamejam game over snelheid, spanning, en russische roulette.

Projectinformatie

Dit is mijn gamejam game die ik met een paar andere klasgenoten heb gemaakt.

We hadden 8 weken en het moest speelbaar zijn via WebGL op itch.io. Het thema was "instantly hooked", dus we dachten aan een snelle Russian Roulette game waarin de speler direct op scherp staat.

Belangrijke Code Voorbeelden

public void PullTrigger()
{
    if (chamber[currentChamber] == true)
    {
        player.Die();
        PlaySound("bang");
    }
    else
    {
        PlaySound("click");
    }

    currentChamber = (currentChamber + 1) % 6;
}

IEnumerator ShakeUI()
{
    Vector3 originalPos = uiElement.localPosition;
    float elapsed = 0f;

    while (elapsed < duration)
    {
        float x = Random.Range(-1f, 1f) * intensity;
        float y = Random.Range(-1f, 1f) * intensity;

        uiElement.localPosition = new Vector3(x, y, originalPos.z);
        elapsed += Time.deltaTime;

        yield return null;
    }

    uiElement.localPosition = originalPos;
}

Behind The Scenes

wij hebben veel problemen gehad met de turn system omdat de tegenstander nooit zijn turn kreeg. We hebben dit opgelost door een coroutine te gebruiken die de beurt van de tegenstander na een korte vertraging activeert. Ook hadden we problemen met de UI die niet goed schudde bij spanning, dit hebben we opgelost door een aparte functie te maken die de UI element tijdelijk verplaatst.
